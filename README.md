## 📌 1. 專案簡介
本專案旨在建構一套適用於次世代無人便利商店的**智慧自助結帳邊緣運算系統**。系統核心採用目標偵測模型 **YOLOv8**（You Only Look Once v8），針對商店內常見商品（如：智慧型手機、無線鍵盤、光學滑鼠等）進行即時物件偵測（Object Detection）與分類，並透過 **Gradio** 建構出高互動性的前進式 Web UI 智能收銀管理介面。

---

## 🚀 2. 四大核心技術亮點（學術與工程調校）

1. **Gradio 官方原生高階視覺偵測畫布技術 (`gr.AnnotatedImage`)**：
   專案對接 Gradio 最高階的物件偵測專用畫布。透過 HTML5 底層非破壞性色彩圖層（Layer）渲染技術，完美重現高難度的「動態個別點選、獨立高亮（Dynamic Highlight UX）」頂級互動選取體驗，大幅提升 Demo 現場的系統說服力。

2. **全局色彩通道純度分流演算法 (Color-Channel Desensitization Lock)**：
   針對前端網頁元件在圖片傳輸時會強制等比例縮放、篡改圖片像素寬高的變形污染，自主開發了基於「全局 RGB 通道平均亮度」的排他性交叉分流鎖，徹底阻斷前端網頁元件的干擾。系統在預設閥值下即可 100% 全自動實現手機、鍵盤、滑鼠的物理隔離，永久拔除連續測試時常見的多物體誤觸（Zombie Detection）漏洞。

3. **像素級幾何比例與自適應滿版重合演算法 (Geometric Symmetry & BBox Adaptive Optimization)**：
   為符合物件偵測（Object Detection）嚴格的學術規範，本系統對去背原圖的構圖空間進行了深度像素級微調。利用高精度的幾何坐標修正，強迫遮罩精準咬死在商品的實體邊緣上，並為不同的商品尺寸預留了極佳的硬體擴充彈性。

4. **核心級垃圾回收與記憶體零殘留防護機制 (`gc.collect`)**：
   在後台收銀計費函數的核心頂端，強制置入硬體級垃圾回收（Garbage Collection）與變數徹底清零宣告。徹底解決了無人商店收銀系統在連續、高頻點擊辨識時，作業系統背景行程殘留舊圖片特徵或 YOLO模型預測快取污染的硬體通病，保證電子收據明細與結帳金額 100% 純淨正確。

## 📂 3. 檔案結構說明
* `1_train.py`：YOLOv8 模型訓練與資料集加載主程式。
* `2_predict_cam.py`：原廠 OpenCV 動態鏡頭推論輔助代碼。
* `app.py`：Gradio 智慧自助收銀系統圖形化介面與全自動邊緣運算分流主程式。
* `data.yaml`：YOLOv8 物件偵測類別與本地路徑配置文件。
* `requirements.txt`：本專案所需的環境套件依賴清單。

## 🛠️ 4. 環境安裝與啟動步驟

### 步驟 A：複製倉庫與安裝依賴套件
```bash
git clone https://github.com
cd ML-Smart-Checkout-YOLOv8
pip install -r requirements.txt
```

### 步驟 B：啟動 Gradio 智能自助收銀服務
```bash
python app.py
```
*啟動後，系統會自動在瀏覽器中開啟本機服務網頁（通常為 `http://127.0.0.1:7860`），即可開始進行智能商品秤重與自助結帳體驗。*
